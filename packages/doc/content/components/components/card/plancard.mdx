# PlanCard

Gympass `<PlanCard />` component is used to display content related to an user's
plan.

```jsx type=highlight
<PlanCard>
  <PlanCard.Tag />
  <PlanCard.Content>
    <PlanCard.Subtitle />
  </PlanCard.Content>
  <PlanCard.List>
    <PlanCard.ListItem />
    <PlanCard.ListItem />
  </PlanCard.List>
  <PlanCard.Actions />
</PlanCard>
```
