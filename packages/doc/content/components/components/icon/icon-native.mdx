### Basic usage

```javascript type=expo
<>
  <Icon as={Accessibility} />
</>
```

### Usage with custom size, fill and stroke

```javascript type=expo
<>
  <Icon
    as={AlertCircle}
    width="medium"
    height="medium"
    fill="energy"
    stroke="energy"
  />
  <Icon as={Home} size="medium" fill="hope" />
</>
```

### Usage with system props

```javascript type=expo
<>
  <Icon
    as={AlertCircle}
    size="medium"
    margin="medium"
    fill="energy"
    stroke="energy"
  />
  <Icon as={Home} size="medium" fill="hope" />
</>
```
