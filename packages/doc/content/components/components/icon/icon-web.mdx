### Basic usage

```javascript
<Icon as={Accessibility} />
```

### Usage with custom size, fill and stroke

```javascript
<>
  <Icon
    as={AlertCircle}
    width="medium"
    height="medium"
    fill="energy"
  />
  <Icon as={Home} size="medium" fill="hope" />
</>
```

### Usage with system props

```javascript
<>
  <Icon
    as={AlertCircle}
    size="medium" 
    margin="medium"
    fill="energy"
  />
  <Icon as={Home} size="medium" fill="hope" />
</>
```
