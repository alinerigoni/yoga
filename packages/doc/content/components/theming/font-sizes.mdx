## Font-Sizes

Font sizes define the size of a typography and its content hierarchy to be
semantic and consistent.

### Usage

This is a simple usage:

```javascript type=highlight
import styled from 'styled-components';
import { theme } from '@gympass/yoga';

const MyCoolComponent = styled.div`
  font-sizes: ${theme.fontSizes.small}px;
`;
```
